## 快速入门5步法

### 第1步：注册账号（1分钟）

1. 访问 www.coze.cn
2. 注册账号（手机号/邮箱）
3. 完成实名认证
4. 进入工作台

### 第2步：选择工作流（2分钟）

**访问GitHub仓库**
```
https://github.com/GongLingRui/coze-hot
```

**浏览工作流目录**
- 按类别选择
- 查看效果预览
- 阅读使用说明

**推荐新手从这些开始**
- 书单推广类（容易变现）
- 治愈系内容（受众广泛）
- 实用工具类（需求稳定）

### 第3步：导入配置（1分钟）

**方法1：手动配置**
1. 查看节点信息详情.txt
2. 在Coze创建新工作流
3. 按说明添加节点
4. 配置参数

**方法2：一键导入**（如果支持）
1. 复制配置链接
2. 在Coze导入
3. 确认配置
4. 保存工作流

### 第4步：测试运行（1分钟）

**准备测试内容**
- 书单视频：准备1本书名
- 治愈视频：准备一个主题
- 工具类：准备测试素材

**运行工作流**
1. 点击运行按钮
2. 输入测试内容
3. 等待生成（1-5分钟）
4. 查看结果

### 第5步：导出使用（随时）

**导出到剪映**
1. 生成完成后导出
2. 打开剪映找草稿
3. 微调优化
4. 导出发布

## 新手常见问题

### Q1: 需要付费吗？
A: Coze基础功能免费，部分高级功能需要付费。

### Q2: 需要编程吗？
A: 完全不需要，拖拽式操作。

### Q3: 生成需要多久？
A: 通常1-5分钟，复杂工作流可能10-15分钟。

### Q4: 可以商用吗？
A: 可以，但需遵守平台规范。

### Q5: 效果不满意怎么办？
A: 调整参数或提示词，多次尝试。
