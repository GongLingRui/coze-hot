# Coze 工作流合集

## 项目简介

这是一个基于 **Coze AI 平台**的强大工作流合集库，专注于自动化视频内容创作。本项目整合了多种 AI 服务（如剪映/CapCut、Vidu、海螺插件等），提供从文本到视频的完整自动化解决方案，特别适合短视频创作者、营销人员和内容创作者使用。

## 技术架构

### 核心平台
- **Coze AI** - 主要工作流编排平台
- **剪映/CapCut** - 视频编辑与渲染
- **Vidu AI** - 图像转视频生成
- **豆包大模型** - 文本生成与理解
- **海螺插件** - 图像与视频处理增强

### 工作流组件
每个工作流目录包含：
- **draft_xxx/** - 工作流草稿文件
- **节点信息详情.txt** - 工作流节点配置（JSON 格式）
- **.png/.jpeg** - 工作流界面截图
- **集成链接** - Coze 到剪映的 API 集成

### 典型工作流结构
```
输入 → 大模型(L L M) → 图像生成 → 语音合成(TTS) → 剪映合成 → 输出
```

## 使用指南

### 前置要求
1. 注册 Coze 账号 (www.coze.cn)
2. 安装剪映/CapCut 应用
3. 准备相应的 AI 服务 API 密钥（如需要）

### 基本使用步骤
1. 选择所需的工作流目录
2. 查看目录中的截图和节点信息
3. 在 Coze 平台导入对应的工作流配置
4. 根据需要修改参数和提示词
5. 运行工作流生成内容

## 特色功能

### 1. 一键导出剪映
多个工作流支持一键导出到剪映，直接在剪映中编辑和渲染。

### 2. 批量处理
支持批量生成视频、图片上传、数据提取等操作。

### 3. 人物一致性
短视频人物一致性功能，确保多镜头中角色外观统一。

### 4. 多平台集成
支持小红书、抖音、淘宝、闲鱼等平台的数据获取。
